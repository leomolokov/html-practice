<blockquote> This block is used to highlight long citations inside a document. Text marked with this tag, will traditionally be shown as a straightened up block with left and right tabulations (around 40 pixels) and also with empty spaces above and below.
Tag <div> is considered as a universal block container and applied in those cases, when the block elements without any additional features are needed.

The <h1>,...,<h6> group of tags defines text headers of different levels, which show a relative importance of a section (paragraph) placed after a header.

<hr> draws a line, the look of which depends on used attributes. The line always starts at a new string and all the elements after are displayed on a next line.

<p> defines a text paragraph. 

			<pre> sets a block of an pre formatted text. Such a text is usually displayed  with a mono shrift and with all the spaces between words. In HTML any number of spaces going in a row on a webpage are shown as ONE. <pre> tag allows to avoid such a things and show text as so it was meant by a developer.
The next tags are not supposed to be placed inside of a container: <pre>, <big>, <img>, <small>, <sub>, <sup>.
