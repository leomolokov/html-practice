A peculiarity of such elements is that depending on the context they can be used as either BLOCK or STRING elements.

<del> tag is used to highlight the text, that was deleted in a new document version. A detailed formatting allows to track the changes inside a text. Browsers usually highlight a text in a <del> container as crossed. See an EXAMPLE.

<ins> tag is used to highlight a fresh written text and usually used along with <del> tag. Browsers usually highlight the text inside a <ins> tag as underscored. See an EXAMPLE.
