The difference between BLOCK and STRING elements is further:

  • String elements can only contain data or other string elements, while block elements can be put into other block elements, string elements and also data. In other words, string elements cannot contain block elements.
  • Block elements are always started from a new line, while string elements aren't highlighted this way.
  • Block elements take all the width space given (from a browser i.e.) while a string elements width equals to width of their content plus the values of spaces, fields and borders.
