A table consists of rows and columns that can contain text amd images. Usually tables are used to organize and represent tabular data.

<table> tag functions as an elements container that define table contents. Any table consists of rows and columns that are set with the tags <tr> and <td>.

<td> tag creates one table cell. <td> tag must be placed into <tr> container which is to be placed inside a <table> tag.

<th> tag is used to create one table cell that is denoted as a heading. Text inside a cell is usually displayed by a browser with a bold shrift and centered.

<tr> tag is a container to create a table row.
