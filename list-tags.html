List is an interconnected set of individuals phrases or sentences that begin with a bullet or a number. Lists provide an opportunity to structure and systemize different data and present them in a representative and user-friendly form.

<ol> tag defines a numbered list, that means that every list element begins with a number or letter and iterates positively. For example.
To buy list:

  1. veggies
  2. fruits
  3. milk

<ul> tag creates a marked list, each element of which begins with a small symbol - marker. An example:
TO DO LIST:

  • write this training webpage until the end
  • commit it
  • push it to github

<li> tag defines a separate list element. While the <ul> or <ol> tag sets a list type. The <li> tag was used in previous examples.

<dd>, <dt>, <dl>: the three tags to create definition lists. Each list of those types begins with a <dl> container, where <dt> tag is to put in, which is creating a definition and <dd> tag, which sets an explanation to the created definition.
